# Retail-Sales-data-Analysis
Understanding Customer behaviour and Product performance for a large retail chain
The dataset includes:
• OrderID: A unique identifier for each order.    
• ProductID: A unique identifier for each product. 
• Category: The product category (e.g., 'Electronics', 'Apparel', 'Home Goods'). 
• Quantity: The number of items sold in that transaction. 
• UnitPrice: The price per unit of the item. 
• CustomerID: A unique identifier for each customer. 
• Date: The date of the transaction.

Using pandas performed a detail data analysis to find out the sales trends over time, best selling categories, average order value etc... The analysis is performed on jupyter notebook, affter data preprocessing,
